public function track(event:String, properties:Object = null):void

Record an event for the user. Optionally, you may send key-value pairs to be associated with the event.

Parameters

Examples
Common usage 
 
     // setup
     KissMetrics.setFactory( FlexMetrics, "kissApiKey" );
     var kiss:IKissMetrics = KissMetrics.getInstance();
     kiss.person = "myid";
     
     // recording metrics
     kiss.track( "event 1" );
     kiss.track( "event 2", {gender:"male"} );
